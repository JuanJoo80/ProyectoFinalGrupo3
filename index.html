Notas Adicionales

Alpine Linux es una distribución ligera basada en "musl libc" y "BusyBox", ideal para contenedores pequeños y seguros, pero con algunas limitaciones:

  • Aplicaciones complejas como Atom, Eclipse o LibreOffice pueden no ser compatibles
  • Muchas aplicaciones están compiladas para sistemas con "glibc" (Debian, Ubuntu)
  • Para aplicaciones con muchas dependencias, considera usar imágenes base como Ubuntu o Debian
